https://node-ts-graphql-playground.herokuapp.com/

## Requirement:

- install yarn
- install node (v12+)

## Testing and run:

```
$ yarn

// development
$ yarn run dev

// production
$ yarn run start

// run test case
$ yarn run test

// use eslint and prettier to format code
$ yarn run lint
```

open localhost:4000
